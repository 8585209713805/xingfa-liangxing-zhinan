# 无依据罪名量刑指引页 · 统一格式规范

> 适用范围：**凡国家及江西省均无专门量刑起点/增量细则的罪名**（如骗取出口退税罪、虚开增值税专用发票罪等危害税收征管类）。
> 有最高法《关于常见犯罪的量刑指导意见》（法发〔2021〕21号）/ 量刑指导意见（二）+ 江西省实施细则（赣高法〔2022〕115号、赣高法〔2025〕20号）作起点依据的常见犯罪（交通肇事、盗窃、诈骗、故意伤害、抢劫、寻衅滋事等）**不适用本规范**，应保留法定起点并标注依据。
> 参考样板：`crimes/骗取出口退税罪量刑指引（江西）.html`（本规范即以其为准克隆）。

## 核心原则
1. **不预测量刑起点**：无细则则只列《刑法》条文与司法解释明确的**法定刑幅度**与**数额档位**，绝不写"拘役–X年"之类的推定起点。
2. **推演内容一律标注非法定依据**：估算器、基准刑增量表里的"起点/增率/比例"仅为内部推演基准，页面须明确声明"非法定依据、实际以法院裁判为准"。
3. **编制依据说明框**置于 overview 之后、`<!-- 法条依据 -->` 之前（标准化 HTML，见下）。

## 各区块统一写法

### 1. 顶部副标题（sub）结尾统一表述
```
本省及国家暂无本罪专门量刑起点细则，故本表不预测量刑起点，仅列法定刑幅度与数额档位
（详见下方"编制依据说明"）；凡涉量刑起点/增量的数值均为推演参考、非法定依据。
```
（前面照常列：刑法第X条、法释〔2024〕4号等依据 + 本罪分档标准）

### 2. 编制依据说明框（标准化，仅改刑法条号）
```html
  <!-- 编制依据说明 -->
  <div style="margin:14px 0;padding:12px 14px;border-left:4px solid #e0a100;background:#fffaf0;border-radius:8px;font-size:13px;color:#7a5a17;line-height:1.75">
    <b>📌 编制依据说明：</b>本页依据《刑法》第XXX条与法释〔2024〕4号整理。本罪<b>无国家及江西省专门量刑起点细则</b>，故本表<b>不预测量刑起点</b>，仅列法定刑幅度与数额档位。
    下方「基准刑增量表」「刑期估算器」中的“起点 / 增率 / 比例”等数值，均为<b>内部推演基准、非法定依据</b>；实际量刑由审理法院在法定刑幅度内依法裁量，本页不替代裁判，可在批注区记录个案判断。
  </div>
```

### 3. 第四项：法定刑幅度与数额档位对照
- 标题：`<h2 class="section">四、法定刑幅度与数额档位对照（定罪量刑区间）</h2>`
- 说明（结构说明 + 粗体⚠ + 列法定刑幅度）：
```html
  <p class="note-tip" style="margin:2px 0 10px;">
    <b>结构说明：</b>蓝色块＝<b>第一量刑幅度</b>（基本犯，法定最高 X 年）；红色块＝<b>第二量刑幅度</b>（数额较大/其他严重情节，X 年–X 年）；橙色块＝<b>第三量刑幅度</b>（数额巨大/其他特别严重情节，X 年–无期）。
    <b>⚠ 关于"量刑起点"：</b>本省（江西）及国家目前<b>尚未出台针对本罪的具体量刑起点指导意见</b>。因此本表<b>不预测、不列示量刑起点</b>，仅列《刑法》第XXX条与法释〔2024〕4号明确规定的<b>法定刑幅度</b>与<b>数额档位</b>；具体量刑起点由审理法院在法定刑幅度内根据个案裁量，本页不替代裁判。
  </p>
```
- 表头"量刑起点幅度"列 → 改为"法定刑幅度（依据）"；表体起点单元格填各幅度**法定刑原文**（如"拘役 – 3年（法定幅度）"）。

### 4. 第五项：基准刑增量（推演参考，非法定依据）
- 标题：`<h2 class="section">五、基准刑增量（推演参考，非法定依据）</h2>`
- 说明中增率须写明"推演参考值（无专门量刑细则，非法定依据）"。

### 5. 第十项：刑期估算器
- 标题：`<h2 class="section">十、情景 → 刑期估算器（附量刑建议，辅助，非裁判依据）</h2>`
- 紧随非法定声明（末尾须含"请勿作为依据对外引用"）：
```html
  <p class="note-tip" style="margin:2px 0 10px;">⚠ <b>重要：</b>本省（江西）及国家<b>无本罪专门量刑起点与增量细则</b>，本估算器的"量刑起点""增率""比例"等数值<b>均为内部推演基准，非法定依据</b>。其中起点默认取各幅度<b>法定最低刑</b>，增率/比例可由你依个案设定。本工具仅供办案思路推演，<b>实际量刑以审理法院裁判为准</b>，请勿作为依据对外引用。</p>
```
- 估算器起点按钮（chips）标签格式统一为 **"第N幅度法定刑"**，data-min/data-max 取该幅度法定刑月数的上下界：
```html
<button ... data-min="1" data-max="36" data-tier="1">拘役–3年（第一幅度法定刑）</button>
<button ... data-min="36" data-max="120" data-tier="2">3年–10年（第二幅度法定刑）</button>
<button ... data-min="120" data-max="180" data-tier="3">10年–无期（第三幅度法定刑）</button>
```

## 与"有依据"页面的区别（务必分清）
| 维度 | 无依据（本规范） | 有依据（常见犯罪） |
|---|---|---|
| 量刑起点 | 不预测、不列示 | 依21号/江西细则列法定起点并标依据 |
| 编制依据说明框 | 有，声明无细则 | 有，声明依据文件 |
| 估算器/增量表 | 标注"非法定依据" | 标注"依文件推演" |
| 第四项标题 | 法定刑幅度与数额档位对照 | 量刑起点与法定刑幅度对照（依文件列示） |

> ⚠ 审稿注意：A类（有依据）页面保留的"拘役–1年/3年–4年/10年–12年"等起点，须由律师对照赣高法实施细则原文核验，本工具无法联网逐一确认。
